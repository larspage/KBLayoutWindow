## 🚀 Quick Start for AI Agents

### Read These Documents (In Order):

1. **PROJECT_SUMMARY.md** - 5 min read
   - Understand what you're building
   - See the big picture
   - Identify available modules

2. **ARCHITECTURE.md** - 30-60 min read
   - Deep dive into system design
   - Read specifications for your module
   - Understand dependencies

3. **docs/AGENT_GUIDE.md** - 15 min read
   - Learn the workflow
   - Understand communication protocols
   - See code examples

4. **docs/API.md** - Reference as needed
   - Look up interfaces
   - Check type definitions
   - See usage examples

### Then:

5. Claim a module (see AGENT_GUIDE.md)
6. Create your feature branch
7. Implement according to ARCHITECTURE.md specs
8. Write tests (see AGENT_GUIDE.md)
9. Update docs/API.md with your interfaces
10. Submit PR
