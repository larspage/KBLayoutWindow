# Vial Layer Display

A lightweight, cross-platform desktop application that displays the current active keyboard layer from Vial-compatible mechanical keyboards.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS-lightgrey.svg)
![Python](https://img.shields.io/badge/python-3.11+-blue.svg)

## Features

- 🎹 **Real-time Layer Display**: Shows your current keyboard layer with <150ms latency
- 🔍 **Zoomable Interface**: Adjustable display size for perfect visibility
- 👁️ **Layer Preview**: View all available layers at a glance
- 📌 **Always-on-Top**: Optional floating window that stays visible
- 🎨 **Customizable**: Themes, colors, fonts, and opacity settings
- 💾 **Persistent Settings**: Remembers your preferences across sessions
- 🖥️ **Cross-Platform**: Works on Windows 10/11 and macOS 12+
- 🪶 **Lightweight**: <50MB memory footprint, minimal CPU usage

## Why Use This?

If you use multiple layers on your mechanical keyboard (especially with Vial firmware), it's easy to lose track of which layer you're on. This app provides a persistent, glanceable indicator so you always know your current keyboard state.
